# Relatório Criativo de Data Analytics - Power BI

# 1. Objetivo do Projeto
O propósito deste desafio é a criação de um dashboard com foco em Design de Interface (UI) e Experiência do Usuário (UX). O projeto visa transformar dados complexos em uma interface visualmente atraente e de fácil compreensão, aplicando conceitos de design thinking ao Business Intelligence.

# 2. Conceitos de Design Aplicados
Para este relatório, o planejamento visual seguiu as seguintes diretrizes:
Layout Estruturado: Uso de espaços negativos para evitar a poluição visual e facilitar o foco nos KPIs principais.
Paleta de Cores e Contraste: Escolha de cores harmônicas que destacam informações críticas sem causar fadiga visual ao usuário.

Iconografia: Planejamento de ícones personalizados para representar categorias de dados, tornando a interface mais intuitiva e menos dependente de textos longos.

# 3. Funcionalidades de Analytics
Apesar do foco criativo, o relatório mantém o rigor técnico através de:
Gráficos Não Convencionais: Escolha de visuais que melhor representam a narrativa dos dados, fugindo do padrão básico quando necessário para uma melhor interpretação.
Navegação Dinâmica: Estruturação de menus ocultos e botões interativos que mantêm o relatório limpo e organizado.

# 4. Justificativa de Entrega
Este repositório documenta a estratégia de design e a arquitetura visual do relatório. Devido à ausência de ambiente Windows para a utilização do Power BI Desktop, a entrega foca na capacidade de planejar dashboards de alto impacto visual e funcionalidade analítica, demonstrando domínio sobre os conceitos de Storytelling de Dados exigidos.
